Building the Application

To build the WAR file:

mvn clean package

The WAR file will be located at:

target/webapp-1.0.0.war

Running the Application

For development (using Spring Boot embedded Tomcat):

mvn spring-boot:run

For deployment (using the WAR file):

Deploy the WAR file to any Java application server (Tomcat, Jetty, etc.)
